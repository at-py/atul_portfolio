Tableau Project

  • In this project, we explore the journey from raw, unclean data to polished, insightful visualizations using Tableau.

  • This particular dataset contains AirBnb customer data set which is puclicly available on kaggle.

  • By employing data cleaning techniques, we address inconsistencies, errors, and missing values within the dataset such as removing blank/ redundant columns, thus ensuring the dataset's reliability and accuracy before Visualization.

  • There are several Visualizations here such as -

    1. Avg Price per Bedroom - The prices go up as the number of rooms increase
    2. Distinct count of Bedroom Listings - The number of listings based on the number of bedrooms provided by lister
    3. Avg price vs Zipcode - A top down View of the mapped zipcode area showcasing avg price per Zipcode
    4. Price by Zipcode - A column view of Avg price in a given Zipcode Area
    5. Revenue for Year - Shows the Revenue Generated throught the dates of the year for 2016
